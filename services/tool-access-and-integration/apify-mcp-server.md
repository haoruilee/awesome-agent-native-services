## MCP
**Status:** ✅ Available (official MCP server).

## What It Does
Lets agents discover and run thousands of web data Actors through MCP tools.

## Why It Is Agent-Native
Tool discovery and execution are exposed directly to autonomous agents via MCP with machine-payable options (x402/Skyfire).

## Primary Primitives
- Actor discovery
- Actor execution
- Dataset retrieval
- Paid tool-call support

## Autonomy Model
Agent can discover→run→fetch results end-to-end without dashboard clicks.

## Identity and Delegation Model
API token or payment identity authorizes calls; execution is attributable per actor run.

## Protocol Surface
Remote MCP over URL.

## Human-in-the-Loop Support
Optional approval can be layered outside MCP.

## Why Generic Alternatives Do Not Qualify
Generic scraping APIs lack MCP-native discovery/tool semantics designed for LLM agents.

## Use Cases
Market research, lead extraction, SERP monitoring, and web intelligence ingestion for agent workflows.
