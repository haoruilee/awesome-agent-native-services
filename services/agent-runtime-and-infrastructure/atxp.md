## What It Does
ATXP is a runtime/payment layer for agents to discover and call paid MCP tools using agent wallets.

## Why It Is Agent-Native
Agent identity + wallet + tool invocation + payment settlement are bundled for autonomous agent loops.

## Primary Primitives
- Agent wallet
- Paid MCP tool invocation
- Per-call pricing / authorization

## Autonomy Model
Agents can execute tool calls and settle payments in-band without human checkout.

## Identity and Delegation Model
Wallet-backed agent identity with policy constraints defined by operator budgets.

## Protocol Surface
CLI, SDK, and MCP-compatible service endpoints.

## Human-in-the-Loop Support
Optional budget/policy guardrails.

## Why Generic Alternatives Do Not Qualify
Conventional payment APIs are human checkout centric and not designed for MCP tool monetization.

## Use Cases
Paid search tools, premium data APIs, and metered agent capabilities.
