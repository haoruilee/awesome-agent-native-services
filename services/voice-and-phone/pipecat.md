## What It Does
Pipecat provides low-latency pipelines for speech-to-text, LLM reasoning, tool calling, and text-to-speech so voice agents can conduct real-time calls and conversations.

## Why It Is Agent-Native
- Designed for autonomous voice-agent loops.
- Realtime tool-calling during conversations.
- Headless runtime operation.

## Primary Primitives
- Realtime voice pipeline
- Turn detection
- Tool invocation in-call
- Transport adapters (WebRTC/SIP)

## Protocol Surface
Python SDK, transport plugins, API integrations.

## Why Generic Alternatives Do Not Qualify
Generic telephony APIs alone do not provide full agent reasoning + tool loop orchestration for realtime voice behavior.
